# Sitio estático básico para MINDCARE

Esta PR añade una plantilla de sitio estático (HTML/CSS/JS) para presentar el proyecto MINDCARE.

Qué incluye:
- index.html: página principal con secciones Sobre, Proyectos y Contacto.
- styles.css: estilos base y responsive.
- script.js: manejo básico del menú móvil y formulario demo.

Siguientes pasos recomendados:
1. Revisa los archivos en la rama `site` y abre un Pull Request hacia la rama por defecto cuando estés listo.
2. Si quieres GitHub Pages, mueve los archivos a `/docs` o configura Pages para publicar desde la rama `site` (o fusiona a la rama por defecto y publica desde `main`/`master`).
3. Personaliza contenido: nombre, texto "Sobre", proyectos, email y logo.
4. Integra el formulario con Formspree/Netlify o con el backend PHP si quieres recibir mensajes.

Yo puedo:
- Mover la plantilla a `/docs` si prefieres publicar con GitHub Pages desde la rama por defecto.
- Crear el Pull Request por ti (si me confirmas) o dejar la rama lista para que tú revises.
